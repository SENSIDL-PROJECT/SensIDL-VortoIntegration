Getting Started

How to create a Project

To create a Project select File -> New -> Java Project enter the project name and hit finish. A new Java project is created. After that right click on the project New -> File. Name the file and and end it with the .sidl extension. The *.sidl file is opened in the SensIDL Editor. In the Editor the Sensor will be written in the SensIDL Language.
