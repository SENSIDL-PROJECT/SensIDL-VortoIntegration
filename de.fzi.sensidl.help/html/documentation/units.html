Units

The Units of SensIDL are based on JScience. Therefore there are all Units of JScience available in SensIDL that are all SI-Units and many NonSI-Units. Furthermore the user can add new units.

Note: Some units (for example °C) may be represented by a '?' in the generated Files (probably best recognizable at the comments in the generated Java files). This is so, because JScience converts these units to a String which can't be shown by the Editor. This is the only restriction that happens with these units.
